SONDER is a crowd-sourced experience sharing network to help facilitate connection and experience sharing between university students. The network is built to fill in gaps in university advising by introducing wide range of student perspectives to help inform individual decisions. The project website is linked above.

Wikipedia Speedruns (Ongoing): https://wikispeedruns.com

Wikipedia Speedruns is a webpage game where players try to get from one wikipedia page to another as fast as possible using only links in the wikipedia articles.

The project is currently in a MVP state, and there are a number of big game features will be added very soon.
